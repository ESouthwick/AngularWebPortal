IP Services

1. Click on DNS: Create Zone

* In the DNS_Zone field enter dom######.lab this number should end in 00 like 030500 not 030504 and the number is specific to the user look for an email with the information for vlabs do not use the numbers used in this tutorial.

* Leave the Zone_Owner field empty

* Submit the form

* This will create the DNS Zone for your VMs you will only need one when requesting an IP Address you do not need to create another zone for each one. One DNS Zone can hold several IP Addresses.

2. Verify your zone was created by clicking on the DNS: List Zones

* In RequestedZones field enter My Zones

* Leave the OtherUser field empty

* Submit the form

* This will verify the DNS Zone you just created if you submitted and nothing showed then you need to go back and redo step 1.

3. Click on the Static IP: Request IP Address

* In the RequestedVLAN field enter the VLAN sent to your email.

* In the Hostname field enter the vlab###### this # was sent to you in an email. Ex vlab030502

* Leave the Description and the Address_Owner fields empty

* Submit the form

* You just requested your first IP Address.

4. Go to the Static IP: List Assignments

* In the requestdAddresses select My Addresses

* Leave the OtherUser field empty

* Submit the form

* This shows a table of all the IP Addresses you have and the information to help the company see who the IP Addresses are assigned to. If nothing appears in the table go back to step 3 and request another IP Address.

5. Click on the DNS: create Record

* In the Hostname field enter the vlab######

* In the DNS_Zone enter the DNS Zone you created dom030500.lab

* In the Target field enter the IP Address you requested for earlier that is linked with the Hostname you are using

* In the RecordType field enter A

* Submit the form

* Congrats you just linked your IPAddress with the DNS Zone making the IPAddress usable in a VM

6. Click DNS:List Zone Records

* In the DNS_Zone field enter the DNS Zone you created dom######.lab

* Submit the form

* This is verifying the system has the record we created and that it is attached to the correct IP Address.

* In order to work with Virtual Machines "VMs", we need to create a IP addresses and in order to create IP addresses we need a DNS zone.

* If you are not quite sure what your DNS zone is check your email. There you will be assigned a DNS name and numbers for your vlabs.

* To add in another IP Address in your DNS Zone repeat steps 3, 4, 5, 6 .

* If you messed up on the IP Address request then just click on the Static IP Return IP Address then enter the IP address you would like to return and Submit the form.

* If you messed up on the DNS Zone creation then use the DNS: Delete Zone then enter the DNS Zone you would like to delete and Submit the form.
